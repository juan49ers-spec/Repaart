### Issue #6 — 0 warnings `any` en producción ✅

Reducción completa: **809 → 0** warnings `no-explicit-any` en archivos de producción.

Patrones aplicados en ~50 archivos:
- `catch (e: any)` → `catch (e: unknown)` + `e instanceof Error`

- Callbacks de Ant Design render → tipados con `unknown`
- Window extensions → `Window & { prop?: Type }`

Los 184 warnings `any` restantes son exclusivamente en archivos de test.
